    Projects

    StudiEats

    A desktop GUI application built for my PRG2104 assignment with Scala and ScalaFX that provides affordable meal delivery services for university students. Features include subscription-based meal plans, real-time order tracking, secure payment processing, and comprehensive admin dashboard for user and menu management. Designed to address food security challenges and promote healthy eating habits among students.

    Algorithm Showcase

    A comprehensive suite of implementations demonstrating core computer science algorithms through interactive, visually engaging applications. Features a complete Binary Search Tree with ASCII visualization, a gamified Dijkstra's pathfinding adventure as a pirate treasure hunt, and an intelligent Sudoku generator/solver using backtracking with multiple difficulty levels.

    Pokémon Ga-Olé

    A comprehensive Pokemon battle simulation game implementing object-oriented programming concepts. Features turn-based combat system, multiple Pokemon types with unique abilities, evolution mechanics, and strategic gameplay elements. Built with Java Swing for an engaging desktop gaming experience.

    SVPS

    This was an assignment for my SEG2202 module. It is a Smart Vehicle Performance System (SVPS) and the assignment required us to design a mobile application as well as an in-car-display interface for optimal user experience.

    University Database System

    A comprehensive database management system for university operations including student enrollment, course management, and grade tracking. Implemented with optimized SQL queries and Python backend.

    GoalPost Gazette

    I created a full-stack blog using Node.js, Express, and MongoDB, featuring admin authentication for secure content management. I built this project around my passion for football, giving me a platform to publish and manage football-related posts and match insights. The blog supports creating, editing, and deleting posts, with a clean and responsive design for smooth user experience. Deployed on Render, this project allowed me to gain hands-on experience with backend APIs, RESTful routing, database modeling, authentication flows, and cloud deployment. It strengthened my ability to combine technical development with personal interests, showing how I can take an idea from concept to a fully deployed application.
