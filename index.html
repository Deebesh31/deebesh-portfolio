Features include subscription-based meal plans, real-time order tracking, secure payment processing, and comprehensive admin dashboard for user and menu management.     Algorithm Showcase

    A comprehensive suite of implementations demonstrating core computer science algorithms through interactive, visually engaging applications.     Pokémon Ga-Olé

    A Java-based Pokemon capture simulation game developed as a team project, implementing advanced object-oriented programming principles with proper inheritance, polymorphism, and modular design. Features a secure user authentication system with CSV-based data persistence, strategic Pokemon capture mechanics, and an innovative Golden Sand reward system that incentivizes continued gameplay even after failed attempts. The game demonstrates strong software engineering practices through comprehensive UML-driven development, robust exception handling, and collaborative team-based development following industry best practices.
